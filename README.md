# Age Calculator

A simple web app that calculates your exact age in years and months from your birth date.

## Live Demo
[View Live](https://punxxet.github.io/Age-Calculator/)

Project URL: https://roadmap.sh/projects/age-calculator


## Features
- Clean date picker input
- Calculates exact age in years and months
- Handles edge cases like birthdays later in the month

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure
age-calculator/
├── index.html    # App structure and layout
├── style.css     # Styling
└── script.js     # Age calculation logic

## How It Works
1. Enter your birth date using the date picker
2. Click **Calculate**
3. Your exact age in years and months is displayed instantly
